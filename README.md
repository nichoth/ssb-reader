# ssb-reader

Read SSB messages and pass them to a pull-stream through.

## Usage

```shell
npm start
```

## Installation

### Source code

```shell
git clone git@github.com:VerseApp/ssb-reader.git
cd ssb-reader
```

### Dependencies

With [npm](https://npmjs.org/):

```shell
npm install
```

With [yarn](https://yarnpkg.com/en/):

```shell
yarn
```
